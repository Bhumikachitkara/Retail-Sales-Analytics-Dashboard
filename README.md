# 📊 Retail Sales Analytics Dashboard | Power BI

An interactive **Business Intelligence Dashboard** built in **Power BI** to analyze retail sales performance, customer behavior, regional trends, and profitability. The project transforms raw transactional data into meaningful business insights using **Power Query**, **Data Modeling**, and **DAX**.

---

## 📌 Project Overview

Businesses generate thousands of sales transactions every day, making it difficult to identify trends through raw data alone.

This dashboard was developed to help decision-makers monitor key performance indicators (KPIs), identify high-performing products and regions, analyze customer purchasing behavior, and track overall business growth through interactive visualizations.

The dashboard provides a clean and intuitive interface for exploring sales data and making data-driven decisions.

---

## 🎯 Objectives

- Analyze overall business performance
- Monitor total sales, profit, and orders
- Compare regional and category-wise performance
- Identify top-selling products
- Track monthly sales trends
- Discover customer purchasing patterns
- Build an interactive dashboard for business reporting
